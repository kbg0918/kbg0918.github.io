Project information

  • 제목: Chat Client-Server
  • 인원: 1명
  • 기간: 11월 말 ~ 12월 초
  • 맡은 업무: 설계 및 구현
  • 주요 기능: 채팅 방 개설, 채팅 방 목록, 채팅 방 입장
  • Tool: Dev C++
  • OS: Windows
  • Language: C
  • Version Control: GitHub

Windows환경 Thread-Mutex 기반의 채팅 프로그램입니다. Server를 열면 임계구역을 통해 Client가 서로 채팅을 주고 받습니다. C언어 공부와 네트워크 프로그래밍의 흐름을 공부하기 위해서 개발했습니다.

Thread-Mutex 기반의 채팅 프로그램을 만들면서 동기화, 임계영역, 상호배제, 네트워크 프로그래밍에 대한 개념을 확실히 배울 수 있었습니다.

구조체를 사용해 Client의 정보를 활용하고 배열, RommId 등을 활용해서 채팅 방 생성, 입장, 목록 등 다양한 기능을 구현했습니다.

채팅 프로그램에 다양한 기능을 넣으면서 C언어 구조체, 포인터, 배열 등 기본기를 다시금 다질 수 있었던 좋은 경험이었습니다.

winsock 라이브러리를 활용한 TCP/IP 통신 채팅 프로그램구현했습니다. Winsock은 Server측과 Client측의 로직이 다른데, 필요한 단계들을 구현해보고 공부해 볼 수 있었습니다.

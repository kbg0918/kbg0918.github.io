Project information

  • 제목: 1대1 Q&A Service MODU
  • 인원: 4명
  • 기간: 8월 말 ~ 10월 말
  • 맡은 업무: 팀장, 설계, Back-End, Front-End
  • 주요 기능: 실시간 알림, 실시간 채팅, 커뮤니티, 제안서
  • FrameWork: Spring boot
  • Back-End: Java, WebSocket, WebSocket-Stomp
  • Front-End: Html/Css, JavaScript, Ajax, JQuery
  • Database: AWS(RDS), MySQL, H2
  • Deployment: AWS(EC2)
  • Version Control: GitHub
  • URL: https://modu.hoseo.dev

MODU는 교수와 학생으로 이루어진 Q&A 채팅 서비스입니다. 현대 사회는 시간이 부족합니다. 그렇기에 모르는 것이나 문제를 빠르고 정확하게 해결하여 여가 시간을 보장하기 위해 개발했습니다.

Spring boot로 개발한 웹사이트입니다. Front-End와 Back-End를 함께 구현했고 각각의 어려움을 해결해나가면서 성장해나간 좋은 경험이었습니다.

실시간 알림은 양방향 통신 프로토콜 WebSocket과 비동기 통신 AJAX를 이용해서 구현했습니다.

실시었간 채팅은 양방향 통신을 기본으로 하지만 단방향 통신을 지원하는 WebSocket-Stomp를 사용했습니다. 새로운 기술을 습득하면서 노력했던 과정이 뜻깊었습니다.

풀 스택으로 개발한 첫 도전이라 저에게는 더욱 의미가 있었고, 직접 UI/UX를 구상하고 구현하며 Front에 대해 많이 배울 수 있었습니다.

AWS의 EC2를 통해 인스턴스 생성, 설정, 연결하면서 배포 경험을 해볼 수 있었습니다. 나아가 CI/CD 파이프 라인을 설정하여 자동화 배포도 경험해 볼 수 있었습니다. 이 과정에서 Source, Build, Deploy 그리고 Cloud 컴퓨팅에 대해 공부했고 경험했던 뜻깊은 시간이었습니다.

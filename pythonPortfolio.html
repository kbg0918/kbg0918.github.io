Project information

  • 제목: 날씨 예측 프로그램
  • 인원: 1명
  • 기간: 1달
  • 맡은 업무: 설계 및 구현
  • 주요 기능:동적 크롤링, Classification(분류), Regression(회귀), Clustering(군집) 기법을 활용한 날씨 예측
  • tool: Anaconda
  • Language: Python, R
  • Version Control: GitHub

머신 러닝의 3가지 기법 Classification(분류), Regression(회귀), Clustering(군집)을 활용한 날씨 예측 프로그램입니다. 개발자는 시대에 뒤처지면 안 된다고 생각합니다. 그래서 인공지능 시대에 맞춰 머신러닝, 딥러닝 기법을 공부하기 위해 개발했습니다.

머신 러닝을 위한 데이터를 얻기 위해동적 크롤링기술을 사용하여 웹 페이지의 데이터를 가져왔습니다. 이 과정에서 데이터 전처리, 동적 크롤링, selenium 등의 개념을 배우고 구현해볼 수 있었던 좋은 경험이었습니다.

웹에서Crawling 한 원시 데이터를 전처리하면서 데이터와 전처리 과정의 중요성을 몸소 깨닫는 계기가 되었습니다.

머신 러닝의 3가지 기법을 활용해보면서 sklearn의 neighbors, svm, tree 등 다양한 모델을 사용했습니다. 각 모델마다 학습 정확도가 달라 이해를 위해 시간을 많이 투자했습니다.

학습 모델의 파라미터를 수정하며 정확도를 높였습니다. 높인 정확도로 예측 결과의 신뢰도를 향상시켰고, 그래프를 활용해 시각적으로 표현하였습니다.

머신 러닝 모델을 개발하며 다양한 라이브러리, 알고리즘 등 다양한 스킬 사용했고 이 과정에서 Python 실력을 향상시켰습니다.
